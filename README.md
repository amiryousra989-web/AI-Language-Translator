# 🌍 AI Language Translator

An AI-powered Language Translator built with **Python** and **Streamlit** that enables users to translate text into multiple languages instantly. The application features automatic language detection, text-to-speech conversion, translation history, and downloadable translations through a clean and user-friendly interface.

---

## ✨ Features

- 🌐 Translate text into multiple languages
- 🤖 Auto Detect Source Language
- 🔊 Text-to-Speech (TTS)
- 📥 Download translated text
- 📜 Translation History
- 🗑️ Clear Translation History
- 🎨 Modern and responsive Streamlit interface

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Deep Translator
- Google Text-to-Speech (gTTS)
- Pandas

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/amiryousra989-web/AI-Language-Translator.git
```

### 2. Navigate to the Project Folder

```bash
cd AI-Language-Translator
```

### 3. Install the Required Libraries

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
streamlit run app.py
```

---

## 🎯 How to Use

1. Select the source language (or choose **Auto Detect**).
2. Select the target language.
3. Enter the text you want to translate.
4. Click the **Translate** button.
5. View the translated text.
6. Listen to the translated audio.
7. Download the translated text if needed.
8. View or clear your translation history.
