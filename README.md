# 🌍 AI Language Translator

An AI-powered Language Translator built with **Python** and **Streamlit**. This application allows users to translate text into multiple languages instantly, listen to the translated text using Text-to-Speech, download translations, and view translation history through a clean and user-friendly interface.

---

## ✨ Features

- 🌐 Translate text into multiple languages
- 🤖 Auto Detect Source Language
- 🔊 Text-to-Speech (Audio Playback)
- 📥 Download translated text
- 📜 Translation History
- 🗑 Clear Translation History
- 🎨 Modern and responsive Streamlit interface

---

## 🛠 Technologies Used

- Python
- Streamlit
- Deep Translator
- Google Text-to-Speech (gTTS)
- Pandas

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Language-Translator.git
```

### 2. Move into the project folder

```bash
cd AI-Language-Translator
```

### 3. Install the required packages

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
streamlit run app.py
```
